# TRON MathTracer

Real-time mathematical function visualizer and tracer designed to be interactive, visually stunning, and highly robust.

## Tech Stack
* **Frontend:** React, Three.js, Vite
* **Backend:** Python

## Getting Started

### How to run frontend:
```bash
cd frontend
npm install
npm run dev
```

### How to run backend:
```bash
cd backend
python main.py
```
